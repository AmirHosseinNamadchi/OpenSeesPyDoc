addToParameter
==============

.. py:currentmodule:: opensees

.. py:function:: addToParameter(paramTag, *args)

   In case that more objects (e.g., :doc:`element`, :doc:`section`)
   are mapped to an existing parameter, the function can be
   used to relate these additional objects to the specific parameter:

   :param int paramTag: tag of existing parameter
   :param args: is the same as defined in :doc:`parameter`
