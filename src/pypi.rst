 PyPi
======

Install Python
--------------

* Install `Anaconda`_ 


Install OpenSeesPy
------------------

* To install

   ::

      python -m pip install openseespy

      python -m pip install --user openseespy

* To upgrade

   ::

      python -m pip install --upgrade openseespy

      python -m pip install --user --upgrade openseespy

   
Import OpenSeesPy
-----------------

::

   import openseespy.opensees as ops
